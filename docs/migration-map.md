## 迁移说明

### 整合策略

文档重构采用了以下整合策略：

1. **内容合并**: 相关主题的文档合并为综合指南
2. **章节提取**: 从多个文档提取相关章节组成新文档
3. **信息归档**: 项目管理类文档移至 `docs/archive/`
4. **结构优化**: 建立清晰的三层文档架构

### 如何使用本映射表

1. 在下方的[完整映射表](#完整映射表)中找到您要查找的旧文档
2. 查看"新位置"列，了解内容的去向
3. 点击链接直接跳转到新文档的相关章节
4. 如果文档被拆分到多个位置，所有位置都会列出
