# 🤖 AI PowerShell 智能助手 - 中文使用指南

一个基于本地 AI 模型的智能 PowerShell 命令行助手，支持中文自然语言交互。

## 🌟 核心特色

- **🇨🇳 中文支持**: 完整的中文自然语言理解和文档
- **🔒 安全第一**: 三层安全保护（命令白名单 + 权限检查 + 沙箱执行）
- **🤖 本地 AI**: 隐私保护，所有处理在本地进行
- **🌐 跨平台**: 支持 Windows、Linux 和 macOS
- **📦 模板系统**: 自定义脚本模板管理

## 🚀 快速开始

### 安装

**方式一：自动安装（推荐）**
```bash
# Windows PowerShell
.\scripts\install.ps1

# Linux/macOS
bash scripts/install.sh
```

**方式二：手动安装**
```bash
# 克隆项目
git clone https://github.com/0green7hand0/AI-PowerShell.git
cd AI-PowerShell

# 安装依赖
pip install -r requirements.txt
```

### 启动

```bash
# 使用启动脚本（推荐）
.\start.ps1      # Windows PowerShell
start.bat        # Windows CMD
./start.sh       # Linux/macOS

# 或直接运行
python src/main.py --interactive
```

### 第一次使用

```bash
# 启动交互模式
python src/main.py --interactive

# 输入中文命令
> 显示CPU使用率最高的5个进程
> 列出当前目录的所有文件
> 查看系统信息
```

## 📖 使用示例

### 系统管理
```python
from src.main import PowerShellAssistant

助手 = PowerShellAssistant()

# 中文描述，自动转换为 PowerShell 命令
助手.process_request("显示内存使用超过500MB的进程")
助手.process_request("检查磁盘空间使用情况") 
助手.process_request("列出所有正在运行的服务")
```

### 文件操作
```python
助手.process_request("列出当前目录下的所有.txt文件")
助手.process_request("创建一个名为'备份'的文件夹")
助手.process_request("查找大于100MB的文件")
```

### 网络诊断
```python
助手.process_request("测试到百度的网络连接")
助手.process_request("显示本机所有IP地址")
助手.process_request("查看网络适配器状态")
```

## 🛠️ 常用命令

### 命令行参数
```bash
python src/main.py --help              # 显示帮助
python src/main.py --version           # 显示版本
python src/main.py --interactive       # 交互模式
python src/main.py --command "命令"    # 执行单个命令
```

### 模板管理
```bash
python src/main.py template list       # 列出所有模板
python src/main.py template create     # 创建模板
python src/main.py template edit <id>  # 编辑模板
python src/main.py template export <id> -o file.zip  # 导出模板
python src/main.py template import file.zip          # 导入模板
```

## 🔒 安全配置

编辑 `config/default.yaml`:

```yaml
security:
  whitelist_mode: "moderate"  # strict/moderate/permissive
  require_confirmation: true
  dangerous_patterns:
    - "Remove-Item.*-Recurse.*-Force"
    - "Format-Volume"
```

## ❓ 常见问题

**Q: 如何更新到最新版本？**
```bash
cd AI-PowerShell
git pull origin main
pip install -r requirements.txt --upgrade
```

**Q: 如何配置 AI 模型？**
编辑 `config/default.yaml`:
```yaml
ai:
  provider: "ollama"
  model_name: "qwen3:30b"
  use_ai_provider: true
```

**Q: 命令执行失败怎么办？**
1. 查看日志 `logs/app.log`
2. 检查 PowerShell 版本
3. 验证命令语法
4. 查看安全设置
