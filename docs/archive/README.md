# 文档归档目录

## 归档目的

本目录用于存放已被整合或替代的历史文档。这些文档不再作为项目的活跃文档维护，但保留用于：

1. **历史参考**: 保留项目演进过程中的文档历史
2. **信息追溯**: 在需要时查找旧版本的信息和决策记录
3. **过渡期支持**: 为用户提供30天的过渡期，适应新的文档结构
