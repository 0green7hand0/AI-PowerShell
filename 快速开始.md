# 🚀 AI PowerShell 智能助手 - 快速开始

5分钟快速上手指南，让你立即开始使用 AI PowerShell 智能助手。

## 📋 前置要求

- Python 3.8 或更高版本
- PowerShell Core 7.0+ 或 Windows PowerShell 5.1+
- 4GB RAM（推荐 8GB）
- 2GB 可用磁盘空间

## 🔧 安装步骤

### 方式一：快速安装（推荐）

**Windows:**
```powershell
# 下载并运行安装脚本
Invoke-WebRequest -Uri "https://raw.githubusercontent.com/0green7hand0/AI-PowerShell/main/scripts/install.ps1" -OutFile "install.ps1"
.\install.ps1
```

**Linux/macOS:**
```bash
# 下载并运行安装脚本
curl -fsSL https://raw.githubusercontent.com/0green7hand0/AI-PowerShell/main/scripts/install.sh | bash
```

### 方式二：手动安装

```bash
# 1. 克隆项目
git clone https://github.com/0green7hand0/AI-PowerShell.git
cd AI-PowerShell

# 2. 创建虚拟环境（可选但推荐）
python -m venv venv

# Windows
venv\Scripts\activate

# Linux/macOS
source venv/bin/activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 验证安装
python src/main.py --version
```

## 📦 使用自定义模板



## 🔒 安全设置

### 配置安全规则

编辑 `config/default.yaml`:

```yaml
security:
  # 安全模式：strict（严格）/ moderate（适中）/ permissive（宽松）
  whitelist_mode: "moderate"
  
  # 是否需要确认
  require_confirmation: true
  
  # 危险命令模式
  dangerous_patterns:
    - "Remove-Item.*-Recurse.*-Force"
    - "Format-Volume"
```

### 测试安全检查

```bash
# 尝试执行危险命令（会被阻止）
> 删除所有文件

# 系统会：
# 1. 检测到危险操作
# 2. 显示警告
# 3. 要求确认或阻止执行
```

## ❓ 常见问题

### Q: 如何更新到最新版本？

```bash
cd AI-PowerShell
git pull origin main
pip install -r requirements.txt --upgrade
```

### Q: 如何配置 AI 模型？

编辑 `config/default.yaml`:

```yaml
ai:
  provider: "ollama"  # 或 "llama"
  model_name: "qwen3:30b"
  use_ai_provider: true
```

### Q: 命令执行失败怎么办？

1. 查看日志文件 `logs/app.log`
2. 检查 PowerShell 版本
3. 验证命令语法
4. 查看安全设置
