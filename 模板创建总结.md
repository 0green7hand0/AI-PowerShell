## ✅ 已创建的模板

### 1. 进程管理模板 ⭐ 新增

**文件**: `templates/process_management/process_manager.ps1`

**功能**:
- 列出进程（按 CPU/内存/名称排序）
- 查找特定进程
- 终止进程
- 监控高资源占用进程

**使用示例**:
```
列出CPU占用最高的10个进程
查找chrome进程
终止notepad进程
监控高资源占用进程
```

### 2. 网络诊断模板 ⭐ 新增

**文件**: `templates/network_diagnostics/network_check.ps1`

**功能**:
- 网络连通性测试（Ping）
- DNS 解析检查
- 端口连接测试
- 路由追踪
- 本地网络接口信息

**使用示例**:
```
测试到百度的网络连接
检查DNS解析和80、443端口
诊断网络连接问题，包含路由追踪
```

### 3. 日志分析模板 ⭐ 新增

**文件**: `templates/log_analysis/log_analyzer.ps1`

**功能**:
- 分析日志文件
- 提取错误、警告信息
- 统计日志级别分布
- 搜索特定模式
- 导出分析结果

**使用示例**:
```
分析日志文件，提取所有错误
统计日志中的错误和警告数量
搜索包含特定关键词的日志
```

### 4. 定时任务管理模板 ⭐ 新增

**文件**: `templates/scheduled_tasks/task_scheduler.ps1`

**功能**:
- 创建 Windows 计划任务
- 查看所有任务
- 查询任务详情
- 启用/禁用/删除任务
- 支持多种计划类型

**使用示例**:
```
列出所有计划任务
创建每天9点执行的备份任务
创建每周一三五执行的清理任务
删除指定的计划任务
```
