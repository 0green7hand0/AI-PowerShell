# 模板管理功能说明

## 概述

新增的模板管理功能提供了一个完整的可视化界面，用于创建、编辑、删除和管理 PowerShell 脚本模板。

## 功能特性

### 1. 模板编辑器 (TemplateEditor.vue)

强大的可视化模板编辑器，包含以下功能：

#### 基本信息编辑
- 模板名称
- 分类选择
- 详细描述
- 关键词标签管理

#### 脚本编辑
- 代码编辑器，支持语法高亮
- 插入参数占位符 `{{参数名}}`
- 脚本格式化
- 脚本验证
- 实时统计（行数、字符数）

#### 参数配置
- 可视化添加/删除参数
- 参数类型选择（字符串、数字、布尔值、选择项）
- 必填/可选设置
- 默认值配置
- 参数描述
- 自动检测脚本中的参数占位符

#### 预览测试
- 模板信息预览
- 参数列表展示
- 脚本内容预览
- 模板测试功能

### 2. 模板管理中心 (TemplateManagementView.vue)

完整的模板管理界面，提供：

#### 多种视图模式
- **卡片视图**：直观展示模板信息
- **列表视图**：紧凑的列表展示
- **表格视图**：详细的表格展示

#### 搜索和筛选
- 关键词搜索（名称、描述、关键词）
- 分类筛选
- 实时过滤

#### 统计信息
- 模板总数
- 分类数量
- 筛选结果数
- 自定义模板数

#### 模板操作
- 创建新模板
- 编辑现有模板
- 删除模板（带确认）
- 使用模板生成脚本
- 刷新模板列表

## 使用方法

### 访问模板管理

1. 启动 Web UI
2. 访问 `/template-management` 路径
3. 或在导航菜单中选择"模板管理中心"

### 创建模板

1. 点击"新建模板"按钮
2. 在"基本信息"标签页填写模板信息
3. 在"脚本编辑"标签页编写 PowerShell 脚本
   - 使用 `{{参数名}}` 作为参数占位符
   - 例如：`Copy-Item -Path {{sourcePath}} -Destination {{targetPath}}`
4. 在"参数配置"标签页定义参数
   - 可以手动添加参数
   - 或使用"自动检测参数"功能
5. 在"预览测试"标签页查看和测试模板
6. 点击"保存"按钮

### 编辑模板

1. 在模板列表中找到要编辑的模板
2. 点击"编辑"按钮
3. 修改模板信息
4. 点击"保存"保存更改

### 删除模板

1. 在模板列表中找到要删除的模板
2. 点击"删除"按钮
3. 确认删除操作

### 使用模板

1. 在模板列表中找到要使用的模板
2. 点击"使用"按钮
3. 在弹出的对话框中填写参数值
4. 生成脚本并执行

## API 端点

模板管理功能使用以下 API 端点：

- `GET /api/templates` - 获取模板列表
- `GET /api/templates/:id` - 获取模板详情
- `POST /api/templates` - 创建模板
- `PUT /api/templates/:id` - 更新模板
- `DELETE /api/templates/:id` - 删除模板
- `POST /api/templates/:id/generate` - 生成脚本

## 特色功能

### 1. 自动参数检测
编辑器可以自动扫描脚本内容，检测所有 `{{参数名}}` 占位符，并自动创建参数定义。

### 2. 脚本验证
在保存前验证脚本内容：
- 检查参数占位符是否都有对应的参数定义
- 检查脚本语法（基础检查）
- 提供错误提示

### 3. 实时预览
在编辑过程中实时预览模板信息、参数列表和脚本内容。

### 4. 多视图切换
根据使用场景选择最合适的视图模式：
- 卡片视图适合浏览和选择
- 列表视图适合快速查看
- 表格视图适合详细对比

### 5. 响应式设计
完全适配移动端和桌面端，提供良好的用户体验。

## 最佳实践

### 模板命名
- 使用清晰、描述性的名称
- 例如："文件备份脚本"、"系统资源监控"

### 参数命名
- 使用驼峰命名法：`sourcePath`、`targetPath`
- 避免使用特殊字符
- 保持简洁明了

### 脚本编写
- 添加注释说明脚本功能
- 使用参数占位符：`{{参数名}}`
- 保持代码格式整洁
- 添加错误处理

### 分类管理
- 合理选择分类
- 保持分类体系清晰
- 使用标准分类名称

### 关键词设置
- 添加相关的搜索关键词
- 包含中英文关键词
- 便于快速查找

## 故障排除

### 模板保存失败
- 检查必填字段是否完整
- 验证脚本内容是否有效
- 检查参数定义是否正确

### 参数检测不准确
- 确保使用正确的占位符格式：`{{参数名}}`
- 参数名只能包含字母、数字和下划线
- 手动添加未检测到的参数

### 脚本生成错误
- 检查参数值是否符合类型要求
- 确保必填参数都已填写
- 验证脚本模板是否正确
