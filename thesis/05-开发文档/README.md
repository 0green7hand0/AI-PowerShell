# 开发文档目录

本目录包含 AI PowerShell 智能助手的开发文档，面向后续维护者和二次开发人员。

## 文档列表

### [developer-guide.md](developer-guide.md)
**完整的开发者指南**

包含以下内容：

1. **环境搭建**
   - 系统要求（最低和推荐配置）
   - 依赖安装步骤（Python、PowerShell、Git、Docker）
   - 推荐开发工具（VS Code、PyCharm、命令行工具）
   - 环境配置方法（配置文件、环境变量、AI 模型）
   - 快速启动指南

2. **代码结构**
   - 完整的目录结构说明
   - 各模块职责详解（9 个核心模块）
   - 依赖关系图（Mermaid 图表）
   - 命名规范（文件、代码、变量、测试）
   - 代码组织原则（SOLID 原则）

3. **开发指南**
   - 编码规范（PEP 8、类型提示、文档字符串）
   - 错误处理规范
   - 日志记录规范
   - 代码注释规范
   - 提交规范（约定式提交、分支命名、PR 规范）
   - 测试规范（测试结构、编写规范、覆盖率目标）
   - 文档规范（文档类型、结构、代码示例）

4. **扩展开发**
   - 如何添加 AI 提供商（完整示例代码）
   - 如何添加安全规则（危险命令模式、自定义验证）
   - 如何添加存储后端（数据库存储示例）
   - 如何开发插件（插件系统架构、示例插件）

5. **调试技巧**
   - 日志使用方法（结构化日志、性能日志）
   - 断点调试技巧（pdb、VS Code、条件断点）
   - 性能分析方法（cProfile、line_profiler、memory_profiler、py-spy）
   - 问题排查流程（系统性排查、常见问题）

6. **部署运维**
   - 部署方式（单机、Docker、Kubernetes）
   - 配置管理方法（配置层级、环境变量、验证）
   - 监控告警方案（日志监控、性能监控、告警配置）
   - 备份恢复方法（数据备份、数据库备份）

- **可扩展性**: 清晰的扩展点说明和示例代码

4. 参考 [扩展开发](developer-guide.md#4-扩展开发)，添加新功能
5. 使用 [调试技巧](developer-guide.md#5-调试技巧)，排查问题
