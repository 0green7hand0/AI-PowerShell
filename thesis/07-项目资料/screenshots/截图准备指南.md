## 📸 截图规范

### 1. 技术规范

- **分辨率**: 1920x1080 或更高
- **格式**: PNG (无损压缩)
- **DPI**: 96 DPI (屏幕标准)
- **颜色**: RGB 24位真彩色
- **文件大小**: 单张 < 2MB

### 2. 内容规范

- **清晰度**: 文字清晰可读
- **完整性**: 包含必要的上下文信息
- **标注**: 关键部分添加箭头或高亮
- **隐私**: 不包含敏感信息
- **美观**: 界面整洁、布局合理

### 3. 命名规范

格式: `{类别}-{功能}-{序号}.png`

示例:
- `01-基本功能-启动界面-01.png`
- `02-安全机制-危险命令拦截-01.png`
- `03-配置管理-配置文件编辑-01.png`

## 🎨 截图后期处理

### 1. 标注工具

推荐工具:
- **Snagit**: 专业截图和标注工具
- **Greenshot**: 免费开源截图工具
- **ShareX**: 功能强大的截图工具
- **Paint.NET**: 简单的图像编辑

### 2. 标注规范

**箭头标注**:
- 颜色: 红色 (#FF0000)
- 粗细: 3px
- 样式: 实线箭头

**高亮标注**:
- 颜色: 黄色半透明 (#FFFF00, 50% opacity)
- 形状: 矩形或圆角矩形

**文字标注**:
- 字体: 微软雅黑
- 大小: 14-16pt
- 颜色: 黑色或红色
- 背景: 白色半透明

### 3. 处理步骤

1. **裁剪**: 去除不必要的边缘
2. **缩放**: 调整到合适的尺寸
3. **标注**: 添加箭头和高亮
4. **文字**: 添加必要的说明
5. **保存**: PNG格式，优化文件大小

## 📝 使用说明

### 拍摄流程

1. **准备环境**
   - 清理桌面
   - 设置合适的分辨率
   - 准备测试数据

2. **拍摄截图**
   - 按照清单逐项拍摄
   - 确保内容完整清晰
   - 多拍几张备选

3. **后期处理**
   - 裁剪和调整
   - 添加标注
   - 优化文件

4. **整理归档**
   - 按规范命名
   - 分类存放
   - 备份保存

### 注意事项

1. **隐私保护**: 不要包含真实的用户名、密码、路径等敏感信息
2. **版权问题**: 确保截图内容不侵犯版权
3. **清晰度**: 确保文字清晰可读
4. **一致性**: 保持截图风格一致
5. **备份**: 保留原始截图和处理后的截图
