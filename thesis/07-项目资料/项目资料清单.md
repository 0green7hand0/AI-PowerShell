## 📋 资料分类

### 1. 项目基础信息

#### 1.1 项目概述
- **项目名称**: AI PowerShell 智能助手
- **英文名称**: AI PowerShell Assistant
- **项目类型**: 智能命令行助手系统
- **开发语言**: Python 3.8+
- **目标平台**: Windows, Linux, macOS
- **开源协议**: MIT License
- **项目地址**: https://github.com/0green7hand0/AI-PowerShell

#### 1.2 核心特性
1. **模块化架构**: 高内聚低耦合的6个核心模块
2. **自然语言处理**: 中文/英文到PowerShell命令转换
3. **三层安全保护**: 白名单、权限检查、沙箱执行
4. **跨平台支持**: Windows/Linux/macOS统一支持
5. **本地AI处理**: 隐私保护的本地模型执行
6. **完整日志系统**: 结构化日志和审计追踪
7. **灵活配置管理**: YAML配置系统
8. **现代化CLI界面**: 彩色输出和交互式体验

#### 1.3 技术栈
- **核心语言**: Python 3.8+
- **命令执行**: PowerShell 5.1+ / PowerShell Core 7.0+
- **配置管理**: PyYAML 6.0.1+
- **数据验证**: Pydantic 2.0+
- **日志系统**: structlog 23.1.0+
- **容器化**: Docker 20.10+
- **AI模型**: LLaMA, Ollama, OpenAI
- **测试框架**: pytest 7.0+
