# 演示视频录制指南

## 概述

本文档详细说明如何录制高质量的系统演示视频，包括录制准备、录制流程、后期制作和质量检查。

---

## 一、录制前准备

### 1.1 录制工具选择

**推荐工具**：

**Windows**：
- **OBS Studio**（免费，功能强大）
  - 下载：https://obsproject.com/
  - 优点：免费、开源、功能完整
  - 缺点：学习曲线稍陡
  
- **Camtasia**（付费，易用）
  - 下载：https://www.techsmith.com/video-editor.html
  - 优点：易用、后期编辑强大
  - 缺点：需要付费

**Linux**：
- **SimpleScreenRecorder**（推荐）
- **OBS Studio**
- **Kazam**

**macOS**：
- **QuickTime Player**（系统自带）
- **OBS Studio**
- **ScreenFlow**（付费）

### 1.2 录制设置

**视频设置**：
```
分辨率: 1920x1080 (Full HD)
帧率: 30fps
格式: MP4
编码: H.264
比特率: 5000-8000 kbps
```

**音频设置**（如果需要旁白）：
```
采样率: 44100 Hz
比特率: 192 kbps
格式: AAC
声道: 立体声
```

### 1.3 环境准备

**桌面清理**：
- [ ] 关闭所有不必要的程序
- [ ] 清理桌面图标
- [ ] 关闭通知和弹窗
- [ ] 关闭自动更新
- [ ] 设置"勿扰模式"

**终端设置**：
- [ ] 字体大小：18-20pt
- [ ] 配色方案：高对比度
- [ ] 窗口大小：全屏或最大化
- [ ] 光标：闪烁的竖线

**系统设置**：
- [ ] 关闭屏幕保护
- [ ] 设置电源为高性能
- [ ] 禁用睡眠模式
- [ ] 调整鼠标速度

## 三、录制流程

### 3.1 OBS Studio 配置

**步骤1：安装和启动**
```bash
# 下载并安装 OBS Studio
# 启动 OBS Studio
```

**步骤2：场景设置**
1. 创建新场景："系统演示"
2. 添加来源：
   - 显示器捕获（全屏）
   - 或窗口捕获（终端窗口）

**步骤3：输出设置**
```
设置 → 输出
- 输出模式: 简单
- 视频比特率: 6000 Kbps
- 编码器: x264
- 录制格式: mp4
- 录制路径: [选择保存位置]
```

**步骤4：视频设置**
```
设置 → 视频
- 基础分辨率: 1920x1080
- 输出分辨率: 1920x1080
- 帧率: 30
```

**步骤5：音频设置**（如果需要）
```
设置 → 音频
- 采样率: 44.1 kHz
- 声道: 立体声
- 桌面音频: 默认
- 麦克风: [选择麦克风]
```

### 3.2 录制步骤

**准备阶段**：
1. [ ] 打开OBS Studio
2. [ ] 检查场景和来源
3. [ ] 测试录制（录制10秒测试）
4. [ ] 检查测试视频质量
5. [ ] 准备演示脚本

**录制阶段**：
1. [ ] 清理桌面和终端
2. [ ] 打开终端，调整大小和字体
3. [ ] 点击OBS的"开始录制"
4. [ ] 等待3秒（给后期剪辑留余地）
5. [ ] 按照脚本执行演示
6. [ ] 完成后等待3秒
7. [ ] 点击"停止录制"

**检查阶段**：
1. [ ] 播放录制的视频
2. [ ] 检查画面清晰度
3. [ ] 检查音频质量（如有）
4. [ ] 检查是否有错误或卡顿
5. [ ] 如果不满意，重新录制

### 3.3 录制技巧

**鼠标操作**：
- 移动速度适中，不要太快
- 点击时稍作停顿
- 避免不必要的移动

**键盘输入**：
- 输入速度适中
- 避免输入错误
- 如果输入错误，可以后期剪辑

**时间控制**：
- 每个场景严格控制时间
- 适当停顿，给观众反应时间
- 总时长控制在3-4分钟

**画面稳定**：
- 避免频繁切换窗口
- 保持终端窗口在屏幕中央
- 避免不必要的滚动

## 四、后期制作

### 4.1 视频剪辑

**推荐工具**：
- **DaVinci Resolve**（免费，专业）
- **Adobe Premiere Pro**（付费，专业）
- **Camtasia**（付费，易用）
- **OpenShot**（免费，简单）

**剪辑步骤**：

1. **导入视频**
   - 将录制的视频导入剪辑软件
   - 创建新项目，设置为1920x1080

2. **剪辑视频**
   - 删除开头和结尾的多余部分
   - 删除中间的错误或停顿
   - 调整每个片段的时长

3. **添加转场**
   - 在片段之间添加淡入淡出
   - 使用简单的转场效果
   - 避免过于花哨的效果

4. **调整速度**（可选）
   - 对于较慢的部分，可以加速到1.2-1.5倍
   - 保持关键部分的正常速度

### 4.2 添加字幕

**字幕内容**：
- 每个场景的说明

### 5.2 视频压缩（如果需要）

**使用HandBrake压缩**：
```
1. 下载HandBrake: https://handbrake.fr/
2. 导入视频文件
3. 选择预设: "Fast 1080p30"
4. 调整质量: RF 20-23
5. 开始压缩
```

**压缩目标**：
- 文件大小: < 100MB
- 画质: 保持清晰
- 时长: 3-4分钟

## 九、应急预案

### 9.1 视频无法播放

**可能原因**：
- 编码不兼容
- 播放器问题
- 文件损坏

**解决方案**：
1. 使用VLC Player（兼容性最好）
2. 使用备用视频文件
3. 使用在线版本（如已上传）
4. 使用截图讲解

### 9.2 视频卡顿

**可能原因**：
- 文件太大
- 电脑性能不足
- 投影仪问题

**解决方案**：
1. 使用压缩版本
2. 关闭其他程序
3. 降低播放分辨率
4. 使用精简版视频
