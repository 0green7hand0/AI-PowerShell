## 准备流程

### 第一步: 生成PDF文档

将所有Markdown文档转换为PDF格式。

**方法一: 使用自动化脚本**
```bash
python scripts/generate_pdfs.py
```

**方法二: 手动转换**
参考 `PDF文档/PDF生成指南.md`

**输出位置**: `PDF文档/`

### 第二步: 整理源代码

清理、打包源代码。

**运行脚本**:
```bash
python scripts/prepare_source_code.py
```

**主要操作**:
- 复制源代码文件
- 排除临时文件和缓存
- 清理调试代码
- 更新README
- 创建压缩包
- 创建Git标签

**输出位置**: `源代码/`

### 第三步: 准备其他材料

收集测试报告、演示视频、截图等。

**运行脚本**:
```bash
python scripts/prepare_other_materials.py
```

**主要操作**:
- 收集测试报告
- 收集演示视频
- 收集截图
- 收集性能数据
- 创建材料检查清单

**输出位置**: `其他材料/`

### 第四步: 组织提交目录

创建规范的提交目录结构并打包。

**运行脚本**:
```bash
python scripts/organize_submission.py
```

**主要操作**:
- 创建提交目录结构
- 复制所有材料到对应目录
- 创建提交说明README
- 创建检查清单
- 打包压缩

**输出位置**: `毕业设计-姓名-学号/` 和 `毕业设计-姓名-学号.zip`

## 一键准备

如果所有材料都已准备好，可以使用一键脚本:

```bash
python scripts/prepare_all_submission.py
```

该脚本会依次执行上述所有步骤。

## 质量标准

### PDF文档
- 纸张: A4
- 页边距: 2.54cm
- 字体: 中文宋体，英文Times New Roman
- 字号: 小四 (12pt)
- 行距: 1.5倍
- 中文显示正常
- 图表清晰
- 目录完整

### 源代码
- 无调试代码
- 无废弃代码
- 代码格式规范
- 注释完整
- README详细
- 可以正常运行

### 演示视频
- 格式: MP4
- 分辨率: 1920x1080 或 1280x720
- 时长: 3-5分钟
- 大小: < 100MB
- 画面清晰
- 声音清楚
- 字幕准确

### 压缩包
- 格式: ZIP
- 大小: < 500MB
- 文件命名: 毕业设计-姓名-学号.zip
- 可以正常解压
- 包含所有必需文件

## 常见问题

### Q1: PDF生成失败怎么办?

- 检查是否安装了Pandoc和LaTeX
- 检查Markdown语法是否正确
- 尝试使用Typora等工具手动转换
- 参考PDF生成指南

### Q2: 压缩包太大怎么办?

- 压缩图片
- 降低视频分辨率
- 移除不必要的文件
- 使用更高的压缩率

### Q3: 源代码无法运行怎么办?

- 检查依赖是否安装
- 检查配置是否正确
- 查看错误日志
- 参考README说明

### Q4: 材料不完整怎么办?

- 查看材料检查清单
- 补充缺失的材料
- 咨询指导教师
- 参考示例材料

### Q5: 提交截止日期快到了怎么办?

- 优先准备必需材料
- 可选材料可以省略
- 确保质量而非数量
- 按时提交
