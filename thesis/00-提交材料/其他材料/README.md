### 2. 演示视频

**文件名**: `系统演示.mp4`

## 准备步骤


## 常见问题

### Q1: 视频文件太大怎么办?

使用FFmpeg压缩:
```bash
ffmpeg -i input.mp4 -vcodec h264 -acodec aac -b:v 2M output.mp4
```

### Q2: 如何批量处理截图?

使用Python脚本:
```python
from PIL import Image
import os

for file in os.listdir('screenshots'):
    if file.endswith('.png'):
        img = Image.open(f'screenshots/{file}')
        img = img.resize((1920, 1080))
        img.save(f'screenshots/{file}')
```

### Q3: 如何制作专业的海报?

### Q4: 测试数据不够怎么办?
