## 生成方法

请参考 `PDF生成指南.md` 了解详细的PDF生成方法。

### 快速开始

**使用Python脚本 (推荐):**
```bash
python scripts/generate_pdfs.py
```

**使用Pandoc手动转换:**
```bash
pandoc input.md -o output.pdf --pdf-engine=xelatex -V CJKmainfont=SimSun -V geometry:margin=2.54cm --toc --toc-depth=3
```

## 质量标准

生成的PDF文档应满足以下标准:

### 格式要求
- 纸张: A4
- 页边距: 2.54cm
- 字体: 中文宋体，英文Times New Roman
- 字号: 小四 (12pt)
- 行距: 1.5倍

### 内容要求
- 目录完整且页码正确
- 图表清晰且编号连续
- 中文显示正常无乱码
- 代码格式规范
- 参考文献格式正确

## 注意事项

1. **文件命名**: 使用规范的文件名，便于识别
2. **文件大小**: 如果文件过大，考虑压缩图片
3. **版本控制**: 保留源Markdown文件，便于修改
4. **备份**: 生成后及时备份到多个位置
5. **打印测试**: 如需打印，先打印一页测试效果

## 常见问题

### 中文乱码
- 确保安装了中文字体
- 使用XeLaTeX引擎
- 指定正确的中文字体名称

### 图片缺失
- 检查图片路径是否正确
- 确保图片文件存在
- 使用相对路径

### 转换失败
- 检查Markdown语法是否正确
- 确保安装了所需工具
- 查看错误日志
