- **解决方案**：
  - 实现规则匹配快速路径（<10ms）
  - 使用LRU缓存（命中率65%）
  - 异步处理
  - 模型量化
- **效果验证**：平均响应时间降至1.5秒

#### 5.8.4 Docker沙箱性能开销
- **问题描述**：容器创建和销毁耗时（>2秒）
- **原因分析**：Docker容器启动开销
- **解决方案**：
  - 容器池复用
  - 轻量级镜像
  - 可选启用沙箱

    # 检查缓存
    cached = self._check_cache(user_input)
    if cached:
        return cached
    
    # 规则匹配
    rule_result = self._match_rules(user_input)
    if rule_result:
        return Suggestion(
            generated_command=rule_result,
            confidence_score=0.9,
            explanation="规则匹配"
        )
    
    # AI生成
    ai_result = self.provider.generate(user_input, context)
    
    # 缓存结果
    self._cache_result(user_input, ai_result)

#### B.3 安全引擎验证代码
```python
def validate_command(self, command: str, context: Context) -> ValidationResult:
    # 第一层：白名单检查
    whitelist_result = self.whitelist.check(command)
    if not whitelist_result.is_valid:
        return whitelist_result
    
    # 第二层：权限检查
    permission_result = self.permissions.check(command)
    if not permission_result.is_valid:
        return permission_result
    
    # 第三层：沙箱检查（可选）
    if self.config.sandbox_enabled:
        sandbox_result = self.sandbox.check(command)
        return sandbox_result
    
    return ValidationResult(is_valid=True, risk_level=RiskLevel.SAFE)
```
